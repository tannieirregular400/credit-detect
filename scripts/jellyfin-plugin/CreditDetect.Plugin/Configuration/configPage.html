Credit Detect

Bridges credit-detect (Python) into Jellyfin's native media segment system. Configured items are analysed on a daily schedule; resulting credit segments appear with a "Skip Credits" button during playback.

Prerequisites

  1. Python 3.10+ installed on the Jellyfin server
  2. credit_detect.py available on the filesystem
  3. A trained model file (model_v1.pb)
  4. opencv-python and ffmpeg installed

Configuration

Path to the Python interpreter (default: python3)
Path to credit_detect.py. Leave empty to auto-detect (looks relative to plugin assembly).
Absolute path to model_v1.pb. Required.
Minimum confidence score (0-1) to expose a credit marker to the skip-button UI. Default: 0.3.
Number of items to analyse per scheduled task run. Default: 10.
Days after which an already-analysed item is re-analysed. 0 = never. Default: 30.

Manual trigger

After saving configuration, run the Credit Detection scheduled task from Dashboard → Scheduled Tasks to start processing.
